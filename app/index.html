Chemical Safety App

Interned for

DHI Group International • Hørsholm, Denmark • 1 month

The problem

DHI's chemical engineers found it cumbersome to carry around hefty, oversized Material Safety Data Sheets (MSDS) and wanted to be able to browse through the company's collection of chemical safety sheets quickly and easily on their smart phones.

The solution

I wrote a mobile app prototype in JavaScript using the Appcelerator Titanium SDK and Alloy MVC framework . This combination of frameworks allowed me to get up and running quickly on Android, iOS, and Windows Phone with a single code base. I also wrote a django + PostgreSQL database backend to store, access, and modify the data sheets through a REST API and user-friendly admin interface.

Sleep Survey Web App

Interned for

41Winks Organization • Menlo Park, California • 1 month

The problem

Nearly 70 million Americans suffer from chronic sleep disorders like sleep apnea, insomnia, restless leg syndrome, and narcolepsy. [1]

The solution

I wrote a survey web app for patients and doctors to diagnose potential sleep disorders. A number of Stanford researchers use the app to gather data for further study. The frontent was developed in HTML/CSS/JS with AngularJS as the MV* framework and Topcoat for basic mobile styles. The backend was developed in Python using the Flask web framework.
