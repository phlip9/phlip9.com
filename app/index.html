APB App

Worked for

DHI Group International • Hørsholm, Denmark • 1 month

The problem

DHI's chemical engineers found it cumbersome to carry around hefty, oversized Material Safety Data Sheets (MSDS) and wanted to be able to browse through the company's collection quickly and easily on their smart phones.

The solution

I wrote a mobile app prototype in JavaScript using the Appcelerator Titanium SDK and Alloy MVC framework. This particular combination of frameworks allowed me to get up and running quickly on Android, iOS, and Windows Phone with a single code base. I then wrote a django + PostgreSQL database backend to store, access, and modify the data sheets.
